Top 40 Linux Commands with Practice Examples

Command Description Practice Example
ls List directory contents ls -l /home/user

pwd Print working directory pwd
mkdir Create a new directory mkdir new_folder
rmdir Remove an empty directory rmdir old_folder

cp Copy files or directories cp file1.txt file2.txt

cat Concatenate and display file content cat file.txt
more View file content with pagination more file.txt
less View file content with backward/forward navigation less file.txt
head Output the first part of files head -n 10 file.txt
tail Output the last part of files tail -n 10 file.txt
grep Search for patterns in files grep "search_term" file.txt
find Search for files in a directory hierarchy find /home/user -name "*.txt"

top Display tasks and system resource usage top
kill Terminate processes by PID kill -9 1234

ssh Securely connect to remote hosts ssh user@remote_host

echo Display a line of text echo "Hello, World!"

uptime Show how long the system has been running uptime
history Show command history history

ping Send ICMP ECHO_REQUEST to network hosts ping google.com
